# Expanding Cards Project

An interactive UI component featuring expanding cards with smooth animations, built with HTML, CSS, and JavaScript.

## 📋 Description

This project showcases a modern expanding cards interface where cards expand smoothly when clicked, revealing more content and providing an engaging user experience. Perfect for portfolios, image galleries, or feature showcases.

## ✨ Features

- **Smooth Animations**: CSS transitions for fluid card expansion
- **Responsive Design**: Adapts seamlessly to different screen sizes
- **Interactive Experience**: Click-based card expansion
- **Clean UI**: Minimalist design with focus on content
- **Cross-browser Compatible**: Works on all modern browsers

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, Transitions)
- Vanilla JavaScript (ES6)
